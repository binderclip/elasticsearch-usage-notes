# python and es

首先是一个基础的库 `elasticsearch-py`，基本上是 es API 的封装，可以做很多的事情。

然后是 `elasticsearch-dsl`，建立在 `elasticsearch-py` 之上，更高级一些，有一些模型之类的抽象。
